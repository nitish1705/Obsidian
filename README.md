# Obsidian

A personal study and revision vault for computer science interview preparation, DSA practice, and core subject notes.

## What’s inside

- **CORE** – foundational computer science topics such as Computer Networks and Core Java
- **CODNG** – programming problem-solving notes organized by topic such as Dynamic Programming, Greedy, and Simple Maths
- **DFS** – graph traversal and cycle-detection problems and solutions
- **ATCODER** – competitive programming notes and practice
- **Level-7 DSA** – advanced DSA study material
- **DESIGN PATERN** – design pattern notes
- **CheckList.md** – a study roadmap and revision checklist

## Purpose

This repository appears to be a structured knowledge base for:

- interview preparation
- competitive programming
- revising core CS concepts
- tracking study progress across topics

## How to use this repo

1. Pick a topic folder.
2. Open the relevant markdown note.
3. Review the theory, examples, and code snippets.
4. Use `CheckList.md` to track what you still need to study.

## Notes

- The repository is organized like an Obsidian vault, so each note is a standalone markdown document.
- Some folders are topic-based collections of problem notes rather than source code projects.

## Contributing

If you want to extend the vault, add new markdown notes under the appropriate topic folder and keep the naming consistent with the existing structure.
